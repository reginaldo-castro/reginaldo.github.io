## 🛠️ Stack Tecnológica

Para garantir máxima performance e um código limpo (sem dependências externas pesadas), utilizei:

- **Core:** HTML5, CSS3 moderno (Grid/Flexbox) e JavaScript (ES6+).
- **Ícones:** FontAwesome.
- **Tipografia:** Inter (Google Fonts).
- **Serviços:** Formspree para processamento de formulários.
- **Hospedagem:** GitHub Pages.

---

## 🌟 Destaques do Projeto

### 🧠 Foco em IA e Engenharia
Diferente de portfólios comuns, este projeto destaca pesquisas de **Pós-Graduação em Machine Learning**, especificamente o uso de modelos **BERT** para análise de sentimentos, unindo a engenharia de software tradicional ao estado da arte da IA.

### 📱 Responsividade & UX
- Menu mobile customizado (drawer lateral).
- Timeline de trajetória profissional adaptável para dispositivos móveis.
- Design focado em legibilidade com uso de **Dark Mode**.

### ⚡ Performance
- Site estático otimizado para carregamento instantâneo.
- Uso de `backdrop-filter` para efeitos de transparência sem comprometer a fluidez.
